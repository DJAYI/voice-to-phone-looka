# Como puedo empezar a utilizar el servicio de Telefonía Ip de Voicetophone?

Como puedo empezar a utilizar el servicio de Telefonía Ip de Voicetophone?

Es muy fácil, solo debes rellenar el formulario de contáctenos y solicitar un demo para tus pruebas.

Para el servicio de troncal sip se entregará un saldo pequeño para llamadas y para la central telefónica IP tambien se entrega un demo por 5 dias con Número virtual de USA para pruebas.
